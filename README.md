# SmartClick

## Install

### 安装

```js
  tnpm install smart-click
```

## Usage

### 引入

```js
  import SmartClick from 'smart-click';
```

### 使用

```js
  <SmartClick
    disabled={false}
    enableActive={true}
    activeClassName="is-active"
    activeStyle={{background: '#ddd'}}>
    <div className="mod-btn">按钮</div>
  </SmartClick>
```
